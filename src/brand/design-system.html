Token reference
/* Duration */
--duration-fast:   100ms;   /* micro-interactions: active states, focus rings */
--duration-normal: 200ms;   /* buttons, badges, card hovers */
--duration-slow:   350ms;   /* modals, drawers, hero entrance */
--duration-slower: 500ms;   /* page transitions, large reveals */

/* Easing */
--ease-default: cubic-bezier(0.4, 0, 0.2, 1);    /* general purpose */
--ease-spring:  cubic-bezier(0.34, 1.56, 0.64, 1); /* buttons, icon hovers */
--ease-out:     cubic-bezier(0, 0, 0.2, 1);        /* enter / appear */
--ease-in:      cubic-bezier(0.4, 0, 1, 1);        /* exit / disappear */

Usage Guide

Importing tokens
/* In any HTML page — before component CSS */
<link rel="stylesheet" href="/src/brand/tokens.css">

/* In a CSS file */
@import '/src/brand/tokens.css';

/* Using a token */
.my-button {
  background: var(--color-brand-600);
  border-radius: var(--radius-lg);
  transition: background var(--duration-normal) var(--ease-default);
}
Logo in HTML
<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="/src/brand/logo/favicon.svg">

<!-- Full logo (header, light background) -->
<img src="/src/brand/logo/logo.svg" alt="BD Cloud Academy" height="40">

<!-- White logo (dark headers / hero sections) -->
<img src="/src/brand/logo/logo-white.svg" alt="BD Cloud Academy" height="40">

<!-- Icon only (mobile nav, favicon fallback) -->
<img src="/src/brand/logo/logo-icon.svg" alt="BD Cloud Academy" width="36" height="36">
What NOT to do
⚠️
Logo rules Never stretch, rotate, recolor, or add drop-shadows to the logo. Never place the colored icon on a colored background with insufficient contrast. Minimum clear-space around the logo = half the icon height on all sides.
🚫
Token rules Never use raw hex values in component CSS. Always reference a token. Never create a one-off color that isn't in the palette — add it to tokens.css first.
